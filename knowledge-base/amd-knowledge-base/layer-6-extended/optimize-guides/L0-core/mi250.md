# AMD Instinct™ MI250 Microarchitecture

## Overview

The AMD Instinct MI250 GPU microarchitecture is built on the CDNA 2 architecture, designed for compute-intensive workloads including HPC, AI, and ML applications. The system architecture emphasizes "extreme scalability and compute performance."

## Single Graphics Compute Die (GCD) Architecture

Each MI250 contains two GCDs with the following specifications:

**Interconnects and Memory:**
- AMD Infinity Fabric interfaces on top and bottom connecting to system components
- Each interface supports four Infinity Fabric links operating at 25 GT/sec (50 GB/sec peak bandwidth for 16-wide links)
- Memory controllers on left and right attach HBM2e modules
- Peak memory bandwidth: 1.6 TB/sec per GCD

**Compute Units:**
- 104 active compute units per GCD
- Each CU contains four SIMD units processing 16 data elements per instruction
- Supports 64 work items (wavefront) at peak 1.7 GHz clock frequency
- Peak FP64 performance: 22.6 TFLOPS per GCD
- Includes specialized matrix cores for operations like matrix multiplication

## Peak Performance Table

| Computation Type | FLOPS/CLOCK/CU | Peak TFLOPS |
|---|---|---|
| Matrix FP64 | 256 | 90.5 |
| Vector FP64 | 128 | 45.3 |
| Matrix FP32 | 256 | 90.5 |
| Matrix INT8 | 1024 | 362.1 |

Aggregated peak memory bandwidth across both GCDs: 3.2 TB/sec

## Dual-GCD Package Architecture

Two GCDs within an OAM (Open Accelerator Module) connect via four Infinity Fabric links at 25 GT/sec, providing "200 GB/sec peak transfer bandwidth between the two GCDs" with bidirectional capacity reaching 400 GB/sec.

## Node-Level Architecture

**PCIe Connectivity:**
- MI250 OAMs connect via PCIe Gen 4 x16 links to host systems
- Each GCD maintains independent PCIe x16 connection to host or PCIe switch
- Some platforms offer x8 interfaces, reducing host-to-GPU bandwidth

**Peer-to-Peer Network:**
- Direct GPU-to-GPU data exchange via AMD Infinity Fabric links (16-wide)
- Operating at 25 GT/sec: 50 GB/sec per link (100 GB/sec bidirectional)
- Specific GCD pairs utilize dual XGMI links for enhanced connectivity
