# AMD Instinct™ MI300 Series Microarchitecture

## Overview

The AMD Instinct MI300 Series GPUs are built on the CDNA 3 architecture, engineered for leadership performance in HPC, AI, and ML workloads. These processors excel at extreme scalability and computational performance across diverse deployment scenarios, from individual servers to exascale supercomputers.

## XCD-Level Architecture

AMD introduces the Accelerator Complex Die (XCD) as the fundamental building block containing GPU computational elements and lower cache hierarchy levels.

**Key XCD Components:**
- 40 Compute Units (CUs): 38 active, 2 disabled for yield management
- Four Asynchronous Compute Engines (ACEs) distributing workgroups to CUs
- 32 KB L1 cache per CU
- Unified 4 MB L2 cache shared across all CUs
- Hardware Scheduler (HWS)

The XCD design represents a smaller, more advanced building block compared to MI200 Series dies. Despite having roughly half the CUs, the MI300 Series can integrate 6-8 XCDs via advanced packaging, yielding approximately 40% more computational capacity than MI250X.

## System Architecture

The MI300 Series combines:
- Up to 8 vertically stacked XCDs
- 8 stacks of High-Bandwidth Memory 3 (HBM3)
- 4 I/O dies for system infrastructure
- AMD Infinity Fabric™ technology as the interconnect

## Matrix Core Enhancements

CDNA 3 Matrix Cores provide significant improvements for AI and ML:

| Data Type | Performance vs. MI250X |
|-----------|------------------------|
| FP16/BF16 | 3× improvement |
| INT8 | 6.8× improvement |
| FP8 | 16× vs. FP32 |
| TF32 | 4× vs. FP32 |

## Peak Performance Capabilities (MI300X)

| Computation Type | FLOPS/Clock/CU | Peak TFLOPS |
|------------------|----------------|------------|
| Matrix FP64 | 256 | 163.4 |
| Vector FP64 | 128 | 81.7 |
| Matrix FP32 | 256 | 163.4 |
| Vector FP32 | 256 | 163.4 |
| Vector TF32 | 1024 | 653.7 |
| Matrix FP16 | 2048 | 1307.4 |
| Matrix BF16 | 2048 | 1307.4 |
| Matrix FP8 | 4096 | 2614.9 |
| Matrix INT8 | 4096 | 2614.9 |

**Theoretical peak memory bandwidth:** 5.3 TB per second

## Node-Level Architecture

In typical deployments with dual-socket AMD EPYC processors:
- Eight MI300X OAM modules attach via PCIe Gen 5 x16 links
- GPUs interconnect through seven high-bandwidth, low-latency AMD Infinity Fabric™ links
- This configuration creates a fully connected 8-GPU system
