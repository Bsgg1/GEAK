# AMD Instinct™ MI100 Microarchitecture

## Overview

The MI100 GPU architecture represents AMD's compute-focused design for high-performance computing and AI/ML applications. The system supports up to eight GPUs per node, connected through AMD Infinity Fabric links operating at 23 GT/sec for inter-GPU communication and PCIe Gen 4 x16 links at 16 GT/sec for host connectivity.

## Memory Architecture

The MI100 features four stacks of HBM2 memory totaling 32GB with a 4,096-bit wide memory interface. "The peak memory bandwidth of the attached HBM2 is 1.228 TB/sec at a memory clock frequency of 1.2 GHz."

## Compute Units and Execution

The GPU contains 120 compute units organized into eight Shader Engines, with fifteen compute units per engine. Each compute unit includes four SIMD units processing 16 data elements per instruction, enabling a wavefront of 64 elements.

**Compute Unit Details:**
- 32KB instruction cache per CU
- Up to ten concurrent wavefronts
- 256 vector general-purpose registers (VGPR)
- 800 scalar general-purpose registers (SGPR)
- Maximum 102 scalar registers per wavefront

## Peak Performance Specifications

| Computation Type | FLOPS/CLOCK/CU | Peak TFLOPS |
|------------------|----------------|------------|
| Vector FP64 | 64 | 11.5 |
| Matrix FP32 | 256 | 46.1 |
| Vector FP32 | 128 | 23.1 |
| Matrix FP16 | 1024 | 184.6 |
| Matrix BF16 | 512 | 92.3 |

At peak clock frequency of 1.5 GHz, theoretical maximum FP64 performance reaches 11.5 TFLOPS.
